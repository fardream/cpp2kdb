Get all the data a simple table.

A simple table is type 98 K. It contains an atomic K pointing to a dictionary. The dictionary's keys are the columns, and the values are all the values by column. The function will also get the number of columns and the rows.

Parameters
simple_tableInput simple table.
column_headingOutput, K object containing the column heading
valuesOutput, an list of k object containing by column
number_of_columnsOutput, number of columns.
number_of_rowsOutput, number of rows.

◆ GetStringFromCharVector()

std::string cpp2kdb::accessors::GetStringFromCharVector ( void *  input_char_vector)

Get std::string from a char vector.

In kdb, string is actually char vector, but without the terminating \0. Symbols are \0 terminated strings in C. The function doesn't perform check on the type. It takes the pointer defined by the k and the number of chars and construct a std::string
